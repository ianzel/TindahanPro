# TindahanPro

TindahanPro is a web-based Sari-Sari Store Management System that helps small business owners manage inventory, sales, profit tracking, supplier information, and customer credit.

## Features
- Product Management
- Sales Recording
- Profit Calculation
- Inventory Tracking
- Supplier Management
- Credit (Utang) Tracking
- Reports and Analytics

## Tech Stack
- Backend: NestJS
- Database: MySQL (TypeORM)
- Frontend: Web-based UI
