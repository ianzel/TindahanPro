# Submission Reflection – TindahanPro

During the development of TindahanPro, I learned how to design and implement a real-world business system that addresses common problems in small retail stores.

I applied concepts such as CRUD operations, API development, and database integration using NestJS and TypeORM.

One challenge encountered was handling system errors and ensuring proper communication between frontend and backend. This was resolved through debugging and improving API consistency.

This project helped me understand the importance of financial computations, inventory control, and user-friendly design.

Overall, the project strengthened my skills in full-stack development and problem-solving.
