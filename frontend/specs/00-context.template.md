# Context – TindahanPro

TindahanPro is a web-based Sari-Sari Store Management System designed to assist small store owners in managing daily operations such as inventory, sales, financial tracking, supplier management, and customer credit records.

The system replaces manual record-keeping methods (e.g., notebooks) with a structured digital solution that ensures accuracy, efficiency, and better decision-making.

It operates as a single-store application and focuses on real-world usability in small retail environments.

Key stakeholders include:
- Store owners (primary users)
- Customers (credit transactions)
- Suppliers (inventory sourcing)

The system interacts with:
- A backend API (NestJS)
- A database (MySQL via TypeORM)
- A frontend interface (web-based UI)

Primary goal:
To provide a simple, reliable, and efficient tool for managing sari-sari store operations.
