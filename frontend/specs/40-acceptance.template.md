# Acceptance Criteria – TindahanPro

## Product Management
- User can successfully add a product
- Product appears in product list
- User can edit and delete product

## Sales
- User can record a sale
- Total amount is computed automatically
- Stock is deducted after sale

## Dashboard
- Displays total sales today
- Displays total profit
- Shows low-stock items

## Supplier
- User can add supplier
- Supplier appears in list

## Credit System
- User can record credit
- User can mark credit as paid
- System tracks unpaid balances

## Reports
- System generates daily and monthly reports
- Data reflects correct calculations
